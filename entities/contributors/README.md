# Contributors

This folder holds one page per verified contributor (`@handle.md`). Pages are **created automatically on a contributor's first verified claim** — they are not hand-authored in advance, so this folder starts empty except for this note.

A contributor page is a wiki node: it carries the `contributor` frontmatter (`handle`, `joined`, `verified_claims`) documented in [[frontmatter-spec]], and it accumulates backlinks from every note the contributor improved — a citable, portable reputation with a URL. That standing, plus Cherry per verified claim and earned Ring-1 access, is why a smart researcher contributes (see [[claims/pending/README]]).

To contribute your first claim, start at [[claims/TEMPLATE]].
