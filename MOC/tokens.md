## How to read a token note

Start with the frontmatter ([[frontmatter-spec]]): `nw_grade` / `nw_grade_worst` for liquidity ([[nw-grade]]), `identity` for [[identity-verification]], `transfer` for [[transfer-feasibility]], `contracts` and `exchanges` for the listing map. Then read the durable prose for the *why*. Grades tell you the book is good; transfer tells you if you can reach it — read both, always (see [[mirage-arb]]).
